HTML Input Types

Here are the different input types you can use in HTML:

<input type="button">

The <input type="button"> element creates a clickable button that can trigger JavaScript code.

Example

<input type="button" value="Click Me">

Output

<input type="checkbox">

The <input type="checkbox"> element creates a checkbox allowing multiple selections.

Example

<input type="checkbox" id="agree">
<label for="agree">I agree</label>

Output

<input type="color">

The <input type="color"> element creates a color picker input.

Example

<input type="color">

Output

<input type="date">

The <input type="date"> element creates a date selection field.

Example

<input type="date">

Output

<input type="email">

The <input type="email"> element is used for input fields that should contain an email address.

Example

<label for="email">Email:</label>
<input type="email" id="email" name="email">

Output




<input type="file">

The <input type="file"> element allows users to upload files from their device.

Example

<input type="file">

Output

<input type="hidden">

The <input type="hidden"> element is used to store data that is not visible to users but can be sent with the form.

Example

<input type="hidden" name="userid" value="12345">

Output

(Hidden fields are not visible to the user)

<input type="image">

The <input type="image"> element displays an image that acts as a submit button.

Example

<input type="image" src="submit.png" alt="Submit">

Output

<input type="month">

The <input type="month"> element allows the user to select a month and year.

Example

<input type="month">

Output

<input type="number">

The <input type="number"> element is used for input fields that should contain a numeric value.

Example

<input type="number" min="1" max="10">

Output

<input type="password">

The <input type="password"> element creates a field for password input, which hides the characters.

Example

<input type="password">

Output

<input type="radio">

The <input type="radio"> element creates a radio button, allowing the user to select one option from a group.

Example

<input type="radio" id="male" name="gender" value="male">
<label for="male">Male</label>
<input type="radio" id="female" name="gender" value="female">
<label for="female">Female</label>

Output


Input Restrictions

Input restrictions help control what users can enter into form fields. Below are some common restrictions:

Restriction Type Description Example
Maxlength Specifies the maximum number of characters allowed in an input field. <input type="text" maxlength="10">
Min and Max Defines the minimum and maximum values for numeric inputs. <input type="number" min="1" max="100">
Pattern Specifies a regular expression the input field's value must match. <input type="text" pattern="[A-Za-z]{3}">
Required Makes the input field mandatory. <input type="text" required>
Readonly Specifies that the input field is read-only and cannot be modified. <input type="text" value="Read-only" readonly>
Disabled Disables the input field, preventing user interaction. <input type="text" disabled>
Step Specifies the interval between legal numbers in a number input. <input type="number" step="5">
