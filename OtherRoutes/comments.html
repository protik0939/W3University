HTML Comments

HTML comments are not displayed in the browser, but they can help document your HTML source code.


Add Comments

With comments you can place notifications and reminders in your HTML code:

Examples:

<!-- This is a comment -->
<p>This is a paragraph.</p>
<!-- Comments are not displayed in the browser -->

Output

This is a paragraph.


Note: Comments are not displayed by the browser, but they can help document your HTML source code.


Hide Content

Comments can be used to hide content.

This can be helpful if you hide content temporarily:

Examples:

<p>This is a paragraph.</p>
<!--<p> This is another paragraph </p>-->
<p>This is a paragraph too.</p>

Output

This is a paragraph.

This is a paragraph too.


Note: You can also hide more than one line. Everything between the <!--and the --> will be hidden from the display.


Examples:

Hide a section of HTML code:

<p>This is a paragraph.</p>
<!--
<p> Look at this cool image:</p>
<img border="0" src="pic_trulli.jpg" alt="Trulli">
-->
<p>This is a paragraph too.</p>

Output

This is a paragraph.

This is a paragraph too.


Note:Comments are also great for debugging HTML, because you can comment out HTML lines of code, one at a time, to search for errors.


Hide Inline Content

Comments can be used to hide parts in the middle of the HTML code.

Examples:

Hide a part of a paragraph:

<p>This<!-- great text -->is a paragraph.</p>

Output

This is a paragraph.
