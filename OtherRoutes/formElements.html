HTML Form Elements

This chapter describes the different elements used within the HTML <form> tag.

The <input> Element

The <input> element is used to create various form controls, such as text fields, checkboxes, and buttons.

Examples

<form>
  <label for="username">Username:</label><br>
  <input type="text" id="username" name="username"><br><br>
  <input type="submit" value="Submit">
</form>

Output




The <label> Element

The <label> element defines a label for an <input> element. Clicking the label focuses the corresponding input field.

Example

<label for="email">Email:</label>
<input type="email" id="email" name="email">

Output




The <select> Element

The <select> element is used to create a drop-down list.

Examples

<form>
  <label for="country">Choose a country:</label><br>
  <select id="country" name="country">
    <option value="bangladesh">Bangladesh</option>
    <option value="pakistan">Pakistan</option>
    <option value="southkorea">South Korea</option>
  </select><br><br>
  <input type="submit" value="Submit">
</form>

Output




The <textarea> Element

The <textarea> element is used to create a multi-line text input field.

Examples

<form>
  <label for="comments">Comments:</label><br>
  <textarea id="comments" name="comments" rows="4" cols="30">
  </textarea><br><br>
  <input type="submit" value="Submit">
</form>

Output




The <button> Element

The <button> element creates a clickable button.

Examples

<form>
  <button type="button" onclick="alert('Hello!')">Click Me!</button>
</form>

Output

The <fieldset> and <legend> Elements

The <fieldset> element is used to group related elements in a form. The <legend> element defines a caption for the <fieldset>.

Example

<fieldset>
  <legend>Personal Information</legend>
  <label for="fname">First Name:</label><br>
  <input type="text" id="fname" name="fname"><br><br>
  <label for="lname">Last Name:</label><br>
  <input type="text" id="lname" name="lname">
</fieldset>

Output

Personal Information



The <datalist> Element

The <datalist> element provides a list of pre-defined options for an <input> element.

Example

<input list="browsers">
<datalist id="browsers">
  <option value="Chrome">
  <option value="Firefox">
  <option value="Edge">
</datalist>

Output

The <output> Element

The <output> element represents the result of a calculation or user action.

Example

<form oninput="result.value=parseInt(a.value)+parseInt(b.value)">
  0<input type="range" id="a" name="a" value="50">100 +
  0<input type="number" id="b" name="b" value="50"> =
  <output name="result">100</output>
</form>

Output

0100 + 0 = 100

Summary Table of Form Elements

Element Description
<input> Creates various form controls (text, button, etc.)
<label> Defines labels for form elements
<select> Creates a dropdown list
<textarea> Creates a multi-line text input
<button> Defines a clickable button
<fieldset> Groups related form elements
<legend> Defines a caption for a <fieldset>
<datalist> Provides a list of pre-defined options
<output> Displays the result of a calculation
