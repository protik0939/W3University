HTML Input Attributes

HTML input attributes provide additional information about an input element.


The required Attribute

The required attribute specifies that an input field must be filled out before submitting the form.

Examples

<form>
  <label for="fname">First Name:</label><br>
  <input type="text" id="fname" name="fname" required><br><br>
  <input type="submit" value="Submit">
</form>

Output




The placeholder Attribute

The placeholder attribute provides a hint describing the expected value of the input field.

Examples

<form>
  <label for="name">Name:</label><br>
  <input type="text" id="name" placeholder="Enter your name"><br><br>
  <input type="submit" value="Submit">
</form>

Output




The value Attribute

The value attribute specifies the initial value for an input field.

Examples

<form>
  <label for="name">Name:</label><br>
  <input type="text" id="name" value="Jubair"><br><br>
  <input type="submit" value="Submit">
</form>

Output




The disabled Attribute

The disabled attribute makes the input field uneditable and prevents form submission.

Examples

<form>
  <label for="disabledInput">Disabled Field:</label><br>
  <input type="text" id="disabledInput" value="Cannot edit" disabled><br><br>
  <input type="submit" value="Submit">
</form>

Output




The readonly Attribute

The readonly attribute makes the input field non-editable but still submits data.

Examples

<form>
  <label for="readonlyInput">Read Only:</label><br>
  <input type="text" id="readonlyInput" value="Read Only" readonly><br><br>
  <input type="submit" value="Submit">
</form>

Output




Attribute Description Example
type Specifies the type of input control to display (e.g., text, password, checkbox). <input type="text">
name Defines the name of the input field, which is used to identify form data when submitted. <input type="text" name="username">
value Specifies the initial value for the input field. <input type="text" value="John Doe">
placeholder Provides a short hint that describes the expected value of the input field. <input type="text" placeholder="Enter your name">
required Indicates that the user must fill out the input field before submitting the form. <input type="text" required>
disabled Disables the input field, preventing the user from interacting with it. <input type="text" disabled>
readonly Prevents the user from editing the input field, but the value can still be submitted. <input type="text" readonly>
maxlength Specifies the maximum number of characters allowed in the input field. <input type="text" maxlength="10">
size Specifies the visible width of the input field (in characters). <input type="text" size="20">
pattern Defines a regular expression that the input value must match to be valid. <input type="text" pattern="[A-Za-z]+">
autofocus Automatically focuses the input field when the page loads. <input type="text" autofocus>
autocomplete Specifies whether the browser should automatically suggest values for the input field based on previous entries. <input type="email" autocomplete="on">
min Specifies the minimum value for input fields of type number, range, etc. <input type="number" min="10">
max Specifies the maximum value for input fields of type number, range, etc. <input type="number" max="100">
step Defines the legal number intervals for input fields of type number, range, etc. <input type="number" step="5">
accept Specifies the types of files that can be uploaded in an input type="file" field. <input type="file" accept="image/*">
multiple Allows the user to select multiple files in an input type="file" field. <input type="file" multiple>
checked Specifies that a checkbox or radio button should be selected by default. <input type="checkbox" checked>
readonly Prevents the user from changing the input value while still allowing it to be submitted. <input type="text" readonly>
